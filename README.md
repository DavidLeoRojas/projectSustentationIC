Este proyecto implementa de manera práctica los conceptos teóricos presentados en tres exposiciones del cohorte:

-Reducción de Dimensionalidad: UMAP, t-SNE, PCA

-Análisis Discriminante Lineal (LDA)

-Clustering: K-Means, DBSCAN, Clustering Jerárquico + Métricas ARI y DBI

El objetivo es mostrar dominio conceptual y práctico mediante código funcional, visualizaciones y métricas.

Todas las implementaciones fueron desarrolladas en Python, utilizando bibliotecas como:

-scikit-learn

-matplotlib

-numpy

-umap-learn


instalar dependencias :

pip install numpy matplotlib scikit-learn

pip install umap-learn
